HumanSkinNet: Automatic Skinning Weight
Prediction Network for Human Animation

Abstract

This work presents HumanSkinNet, an end-to-end automatic architecture specifically designed for predicting skinning weights from an input humanoid character model and its associated skeleton. Given a character mesh and skeleton as input, the proposed novel architecture explicitly decouples the body skeleton and the component skeleton, extracting both common body skeletal features and character-specific component skeletal features as attributes of the graph nodes. To achieve smoother weight distribution, the local bone chain is considered as the smallest unit, taking into account the weight transfer between bones in the same chain. Instead of using vertex-bone distances alone, the proposed method involves using the bone selection module (BSM) to dynamically select the affected local bone chains. The BSM can easily be applied to existing models as a plug-and-play module, improving performance significantly. Our contributions have been shown to be effective in experimental results, with HumanSkinNet surpassing current state-of-the-art alternatives.
